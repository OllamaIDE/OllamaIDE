### 💻 What is OllamaIDE?

**OllamaIDE** is an open-source, highly performant native Windows IDE fully written in C# .NET 8 / WPF. We built this ecosystem to provide an autonomous, Cursor-like coding experience without the premium subscription costs, and without relying on wrapped browser technologies like Electron.

Instead of sending your proprietary codebase to remote API endpoints, everything runs **purely localized** connecting directly to your internal [Ollama](https://ollama.com/) instance on port `:11434`.

### ⚡ Key Capabilities

- **Native Architecture**: Written in C# .NET 8 for zero-latency UI loading and hardware-accelerated WPF rendering.
- **Agentic Loop**: Autonomous capabilities to edit, modify files, detect language shifts, and execute terminal commands entirely locally.
- **Deep Syntax Interfacing**: AvalonEdit powers our fast rendering components featuring rich JSON, Python, and C# formatting out of the box with zero setup.
- **Dark Mode Perfect**: No distracting light bars or mismatched scroll-views natively out of the box.
